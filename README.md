# Grover
A best practice for Spring Boot Web.

## 技术栈及依赖
- SpringBoot
- MyBatis
- Lombok
- Maven
- JUnit
- Jackson
- Guava: 工具库（集合、缓存、位图、原语等等）
- Javassist: 动态编辑Java字节码

## 环境
- MySQL 5.7+
- Redis 5.0+


## 配置
